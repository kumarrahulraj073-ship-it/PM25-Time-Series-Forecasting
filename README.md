# PM2.5 Forecasting using Machine Learning and Deep Learning

## Overview

This project presents a comprehensive framework for forecasting PM2.5 concentration using Machine Learning, Deep Learning, and Wavelet Denoising techniques.

The study utilizes CPCB air quality datasets from Patna and Hyderabad and evaluates multiple forecasting models to improve air pollution prediction accuracy.

---

## Objectives

- Forecast PM2.5 concentration levels
- Compare ML and DL forecasting models
- Apply wavelet-based denoising for noise reduction
- Evaluate model performance using standard metrics

---

## Models Implemented

### Machine Learning

- SVR (Support Vector Regression)
- Random Forest
- ARIMA

### Deep Learning

- RNN
- LSTM
- GRU
- BiLSTM
- BiGRU
- CNN
- CNN-LSTM
- GRU-LSTM

---

## Signal Processing

- Discrete Wavelet Transform (DWT)
- Daubechies Wavelet (db4)
- VisuShrink Thresholding

---

## Evaluation Metrics

- RMSE
- MAE
- MAPE
- R² Score

---

## Best Results

### Patna

- Best Model: BiGRU
- R² = 0.974

### Hyderabad

- Best Model: BiGRU
- R² = 0.993
